# Setup

## Requirements

* Python 3.8+

## Installation

Install it directly into an activated virtual environment:

```text
$ pip install vicon_nexus_unity_stream_py
```

or add it to your [Poetry](https://poetry.eustace.io/) project:

```text
$ poetry add vicon_nexus_unity_stream_py
```
